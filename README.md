O projeto consiste em uma aplicação Full-Stack para monitoramento de dispositivos de rede. O back-end foi desenvolvido em Java com Spring Boot, responsável pelo cadastro dos equipamentos, armazenamento no MySQL e execução dos testes de conectividade. O front-end, feito em React, consome a API para exibir os dispositivos, cadastrar novos itens e acionar os testes. 
----------------------
Para executar a aplicação, basta iniciar o servidor Java com ./gradlew bootRun e, no front-end, instalar as dependências com npm install e iniciar o sistema com npm run dev.
